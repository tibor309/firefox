# 🦊 Firefox Stable
This is the [stable][stable_build] version of the firefox browser. This version is updated every month.

## Setup
To use this version of the browser, change the docker image tag to `latest` or `stable`. For setup options and instructions, check the [main page][main].


[stable_build]: https://www.mozilla.org/en-US/firefox/new/
