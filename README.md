# 🦊 Firefox Stable
This is the [stable edition][stable] of the firefox browser. This version is updated every month.

## Setup
To use this edition of the browser, change the docker image tag to `latest` or `stable`. For setup options, check the [main repo][main].


[stable]: https://www.mozilla.org/en-US/firefox/new/
[main]: https://github.com/tibor309/firefox/tree/master
